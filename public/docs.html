üìñ Introduction

PayUs-as-a-Service (PUaaS) is a free API that generates randomized payment reminder messages in various tones. Perfect for developers who need to send past-due invoice reminders but want to maintain different communication styles.

Base URL: https://puaas.sticknologic.is-a.dev
Method: GET
Authentication: None required
Rate Limit: 60 requests per minute per IP

üé≠ Available Tones

Tone Description Best For
Professional Formal, business-like language Corporate clients, B2B invoices
Friendly Warm and approachable Regular customers, long-term clients
Frank Direct and straightforward Clear communication, no-nonsense clients
Funny Humorous and lighthearted Informal relationships, creative industries
Playful Witty and engaging Casual clients, modern businesses

üíª Code Examples

JavaScript / Fetch API

fetch('https://puaas.sticknologic.is-a.dev/payus/professional')
  .then(response => response.json())
  .then(data => {
    console.log('Message:', data.message);
    console.log('Tone:', data.tone);
  })
  .catch(error => console.error('Error:', error));

cURL

curl https://puaas.sticknologic.is-a.dev/payus/friendly

Python

import requests

response = requests.get('https://puaas.sticknologic.is-a.dev/payus/funny')
data = response.json()
print(f"Message: {data['message']}")
print(f"Tone: {data['tone']}")

PHP

$response = file_get_contents('https://puaas.sticknologic.is-a.dev/payus/frank');
$data = json_decode($response, true);
echo "Message: " . $data['message'] . "\n";
echo "Tone: " . $data['tone'] . "\n";
